## 🌈 pipeline 작성



### 🐳 pipeline script

* gradle 설치가 따로 필요없이 가능하다.

```groovy
pipeline {
    agent any

    stages {
        stage('Git Clone 🦊') {
          steps {
            git url: 'https://github.com/heesootory/jenkins_pipe_test.git',
                branch: 'main',
                credentialsId: 'github_token'
            }
        }
        stage('Bulid Gradle 🍀') {
            steps {
                echo 'Bulid Gradle'
                dir('.'){
                    sh './gradlew clean build'
                }
            }
        }
    }
}
```
