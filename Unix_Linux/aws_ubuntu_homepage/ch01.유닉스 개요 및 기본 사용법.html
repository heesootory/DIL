ch01.유닉스 개요 및 기본 사용법



# 유닉스 구조

  • 커널
    • 운영체제의 핵심
    • 컴퓨터 자원 관리
    • 프로세스 관리, 메모리 관리, 파일 시스템 관리, 장치 관리
  • 셸
    • 사용자 인터페이스 제공
    • 사용자의 명령을 입력 받아 처리한 후 결과를 출력
  • 유틸리티
    • 각종 프로그래밍 개발 도구, 문서 편집도구 등

# 명령의 구조

> 명령 [옵션] [인자]

# 기초 명령 사용법

  • banner : 화면에 큰 문자를 출력
  • date : 날짜와 시간의 출력
  • clear : 화면을 깨끗히 지움
  • man : 명령에 대한 설명 출력

# 비밀번호 변경 : passwd

- 전화번호, 생일 등 추측이 가능한 패스워드 사용금지
- 8자 이상으로 숫자나 특수 문자를 포함하여 지정.

# 유닉스 시스템 접속 해제 - logout

  • logout
  • exit
  • Ctrl + D
