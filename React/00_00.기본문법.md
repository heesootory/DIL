## useState

* const [state, func] = React.useState(defaultValue);
    -  state는 항상 func로 수정한다. (직접 수정하지 않음)
    


```js
function MinutesToHours(){
            const [amount, setAmount] = React.useState(0);
            const [inverted, setInverted] = React.useState(false);

            const onChange = (event) => {
                setAmount(event.target.value);
            }
            const reset = () => setAmount(0);
            const onFlip = () => {
                reset();
                setInverted((current) => !current);
            }
            return (
                <div>
                    <div>
                        <label htmlFor="minutes">Minutes</label>
                        <input 
                            value={inverted ? amount * 60 : amount} 
                            id="minutes" 
                            placeholder="Minutes" 
                            type="number"
                            onChange={onChange} 
                            disabled={inverted}
                        />
                    </div>

                    <div>
                        <label htmlFor="hours">Hours</label>
                        <input 
                            value={inverted ? amount : Math.round(amount/60)} 
                            id="hours" 
                            placeholder="Hours" 
                            type="number" 
                            onChange={onChange}
                            disabled={inverted === false}
                            />   
                    </div>

                    <button onClick={reset}>Reset</button>
                    <button onClick={onFlip}>{inverted ? "Turn back" : "Invert"}</button>
                </div>
            );
        }

```

* inverted라는 변수를 setInverted라고 정의한 함수가 변경시킬 것이다.

* setInverted함수가 호출될때마다, return에 있는 html코드들이 리렌더링 된다.

* useState() -> 괄호안의 값은 inverted의 default값이다.


* button onClick={reset}
    - html태그를 찾아서 이벤트리스너를 붙여주는 과정을 단축!

### props로 함수도 보내는게 가능

* 주의할 점
    - 컴포넌트에 적어주는 요소들은 props이다!!!
        - onClick이라고 요소명을 적어줘도, 이벤트 리스너가 아님!!
    - html 태그에 달아주는 onClick이 이벤트 리스너!
    - 따라서, props로 이벤트 리스너 함수를 보내주고, html태그에다 달아줘야 동작가능!


* changeValue라는 함수를 prop로 전달한 예제코드

```js
<!DOCTYPE html>
<html>
    <body>
        <div id="root"></div>
    </body>

    <script src="https://unpkg.com/react@17.0.2/umd/react.development.js"></script>
    <script src="https://unpkg.com/react-dom@17.0.2/umd/react-dom.development.js"></script>
    <script src="https://unpkg.com/babel-standalone@6/babel.min.js"></script>

    <script type="text/babel">
        const root = document.getElementById("root");
        function Btn({text, changeValue}){
            return (
                <button 
                    onClick={changeValue} 
                    style={{
                        backgroundColor : "tomato",
                        color : "white",
                        padding: "10px 20px",
                        border : 0,
                        borderRadius: 10,
                    }}>
                    {text}
                </button>
            );
        }
        
        function App(){
            const [value, setValue] = React.useState("Save Change")
            const changeValue = () =>{
                setValue("Revert Change");
            }
            return (
                <div>
                    <Btn text={value} changeValue={changeValue}/>
                    <Btn text="Countine" />
                </div>
            );
        }
        ReactDOM.render(<App />, root)
    </script>

</html>
```


### props memo

* 컴포넌트가 묶여있을때, 하나가 변경될 때마다 그안의 모든 컴포넌트를 다시 그리게 되면, 앱동작이 느려질 수 있다.


* 따라서, 캐시를 이용하는 느낌으로 변경이 필요하지 않은것은 다시 렌더링하지 않게 메모를 하는 기능이 있다.

* 변경할 컴포넌트만 변경되게 하는 예제코드

```js
<!DOCTYPE html>
<html>
    <body>
        <div id="root"></div>
    </body>

    <script src="https://unpkg.com/react@17.0.2/umd/react.development.js"></script>
    <script src="https://unpkg.com/react-dom@17.0.2/umd/react-dom.development.js"></script>
    <script src="https://unpkg.com/babel-standalone@6/babel.min.js"></script>

    <script type="text/babel">
        const root = document.getElementById("root");
        function Btn({text, changeValue}){
            console.log(text + "was rendered");
            return (
                <button 
                    onClick={changeValue} 
                    style={{
                        backgroundColor : "tomato",
                        color : "white",
                        padding: "10px 20px",
                        border : 0,
                        borderRadius: 10,
                    }}>
                    {text}
                </button>
            );
        }
        const MemorizedBtn = React.memo(Btn);
        function App(){
            const [value, setValue] = React.useState("Save Change")
            const changeValue = () =>{
                setValue("Revert Change");
            }
            return (
                <div>
                    <MemorizedBtn text={value} changeValue={changeValue}/>
                    <MemorizedBtn text="Countine" />
                </div>
            );
        }
        ReactDOM.render(<App />, root)
    </script>

</html>
```


### PropTypes

* prop 의 형을 검사해주는 cdn를 넣고, 검사할 형들을 정리해놓으면, 에러를 던진다.

* isRequired : 꼭 필요한 요소 / 없다면 - 있든없든 상관없음.

* props요소에 기본값을 정해줄 수 있음.

* 예제코드

```js
<!DOCTYPE html>
<html>
    <body>
        <div id="root"></div>
    </body>

    <script src="https://unpkg.com/react@17.0.2/umd/react.development.js"></script>
    <script src="https://unpkg.com/react-dom@17.0.2/umd/react-dom.development.js"></script>
    <script src="https://unpkg.com/prop-types@15.6/prop-types.js"></script>
    <script src="https://unpkg.com/babel-standalone@6/babel.min.js"></script>

    <script type="text/babel">
        const root = document.getElementById("root");
        function Btn({text, fontSize = 12}){
            return (
                <button 
                    style={{
                        backgroundColor : "tomato",
                        color : "white",
                        padding: "10px 20px",
                        border : 0,
                        borderRadius: 10,
                        fontSize,
                    }}>
                    {text}
                </button>
            );
        }
        Btn.propTypes = {
            text: PropTypes.string.isRequired, 
            fontSize: PropTypes.number,
        }
        function App(){
            return (
                <div>
                    <Btn text="Save Changes" fontSize={18} />
                    <Btn text={20} fontSize="contiue" />
                </div>
            );
        }
        ReactDOM.render(<App />, root)
    </script>

</html>
```
