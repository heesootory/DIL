* JSON 내장 매서드
    - JSON을 조작하기 위해 두가지 매서드 제공
    - JSON.parse() : JSON 문자열 -> 자바스크립트 객체
    - JSONstringify() : 자바스크립트 객체 -> JSON 문자열


```js
let student = {
    name : "heesoo",
    age : 20,
    hobby : ["공부", "숙면"],
    "favorite singer" : "아이유",
};

let jsonData = JSON.stringify(student)

jsonData
> 'string'

typeof(jsonData)
>

let parseData = JSON.parse(jsonData)

parseData
> 

typeof(parseData)
> 'object'

```
