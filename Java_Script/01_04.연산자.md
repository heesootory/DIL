# 연산자


## 산술 연산자

* **(거듭제곱)

```js
console.log(3**10)
> 59049

```


## 일치 연산자

```js
var i = 100;
var j = "100";

// 비교시 암묵적 타입 변환을 통해 타입을 일치시킨 후 비교
console.log("i == j", i == j);      
> true          // 값만 확인
console.log("i === j", i === j);
> false         //값과 타입을 확인.

```
