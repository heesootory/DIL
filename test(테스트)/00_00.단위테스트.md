* Junit Platform : 테스트를 실행해주는 런처 제공. TestEngine API 제공.

* Jupiter: TestEngine API 구현체로 Junit 5를 제공.

* Vintage : Junit 3과 4를 지원하는 TestEngine 구현체.

> 따라서, 우리가 사용하는 Junit5는 Jupiter를 사용하게 됨.

```java
import org.junit.jupiter.api.*;
```
